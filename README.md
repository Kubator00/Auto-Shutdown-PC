# Auto-Shutdown-PC
Turn off the computer after specified time.<br/>
Works on Windows OS. 
Program available in English and Polish. <br/>

Different tasks:
 <ul>
  <li>shut down</li> 
  <li>reboot</li>
  <li>logout</li> 
  <li>hibernate</li> 
 </ul>
User can choose if operation will be execute after the entered time or the specified hour.
